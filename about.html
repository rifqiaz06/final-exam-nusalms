Tentang Saya

Hai, perkenalkan nama saya Rifqi Akhmad Zakaria, saya adalah seorang siswa yang saat ini sedang menempuh pendidikan kejuruan di SMKN 1 Adiwerna di Jurusan Teknik Komputer dan Jaringan. Saya membuat website ini dengan tujuan untuk menyajikan portofolio pribadi saya dan memberikan wawasan kepada pengunjung tentang keterampilan, proyek, dan pengalaman saya dalam bidang Web Programming. Melalui halaman-halaman ini, saya berharap dapat memperlihatkan dedikasi saya terhadap Web Programming, serta memberikan gambaran yang jelas mengenai karya-karya saya yang telah diselesaikan dan proyek-proyek yang telah saya rancang. Website ini juga berfungsi sebagai saluran komunikasi untuk potensial klien, mitra kerja, atau sesama profesional yang tertarik untuk berkolaborasi. Dan jika anda tertarik dengan saya bisa hubungi saya lewat LinkedIn dibawah ini
